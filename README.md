# quarkus-camel-multipledb

Sample application for trying out multiple dbs with quarkus

To run:

```
mvn quarkus:dev
```

# Persons

## Persons UI

http://localhost:8080/persons.html

## Persons REST

http://localhost:8080/persons

# Fruits

## Fruits UI

http://localhost:8080/fruits.html
